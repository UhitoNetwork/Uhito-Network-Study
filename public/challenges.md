# Practice Challenges & CTFs

Sharpen your skills with these free platforms:

| Platform | Type | Description |
|----------|------|-------------|
| [TryHackMe](https://tryhackme.com) | Hands-on labs | Beginner-friendly rooms, guided paths |
| [HackTheBox](https://www.hackthebox.com) | CTF/machines | Realistic penetration testing challenges |
| [OverTheWire](https://overthewire.org) | Wargames | Learn security concepts through games |
| [PicoCTF](https://picoctf.org) | CTF | Great for beginners, run by Carnegie Mellon |
| [CTFtime](https://ctftime.org) | CTF tracker | Find upcoming CTFs and teams |
| [Root-Me](https://www.root-me.org) | Challenges | 400+ challenges in various categories |
| [VulnHub](https://www.vulnhub.com) | Vulnerable VMs | Download and hack locally |
| [PortSwigger Web Security Academy](https://portswigger.net/web-security) | Free labs | Web app security (Burp Suite) |
| [CryptoHack](https://cryptohack.org) | Crypto challenges | Learn cryptography by solving puzzles |

---

Start with TryHackMe if you're new – it walks you through everything.
