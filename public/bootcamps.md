# Free & Affordable Bootcamps

| Name | Cost | Description |
|------|------|-------------|
| [100Devs](https://100devs.org) | Free | Full-stack web dev with focus on networking and community |
| [The Odin Project](https://www.theodinproject.com) | Free | Web development (full stack) |
| [FreeCodeCamp](https://www.freecodecamp.org) | Free | Certifications in responsive web, JS, Python, etc. |
| [CS50](https://cs50.harvard.edu) | Free | Harvard's intro to computer science |
| [Cybrary](https://www.cybrary.it) | Free tier | Cybersecurity courses and labs |
| [TryHackMe](https://tryhackme.com) | Free tier | Hands-on cyber security rooms |
| [HackTheBox Academy](https://academy.hackthebox.com) | Free modules | Penetration testing, ethical hacking |
| [SANS Cyber Aces](https://www.cyberaces.org) | Free | Introductory cybersecurity courses |

---

Most bootcamps also offer scholarships – always check the website.
